While Loop Practice

## 📌 Objective
To master looping logic and improve control over iteration-based problems.

## 🧠 How I Practice

1. I start with basic loop structure:
   - Printing numbers
   - Printing even/odd numbers
2. Then I solve logic-building problems:
   - Sum of digits
   - Reverse a number
   - Palindrome check
3. After that, I practice condition-based loops:
   - Infinite loop with break
   - Menu-driven systems
4. Finally, I combine while loop with:
   - Conditional statements
   - User input
   - Nested loops

## 🎯 What I Focus On

- Loop condition accuracy
- Avoiding infinite loops
- Proper use of break and continue
- Logical thinking improvement
